# myfirstreact

ReactJS-HOL-1 : Set up a React environment using create-react-app.
Prints "welcome to the first session of React" as the heading of the page.

## Steps followed
1. Installed Node.js and NPM
2. Installed create-react-app globally
3. Created app "myfirstreact" using: npx create-react-app myfirstreact
4. Navigated into the project folder
5. Opened project in Visual Studio Code
6. Replaced content of App.js with the required heading
7. Ran the app using: npm start
8. Opened localhost:3000 in the browser
